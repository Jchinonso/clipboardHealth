# Clipboard health
## Stack for project
1. React JS 
2. NextJs
3. Tailwind CSS
4. Jest
 

## Installation Instructions
### Run App in a developement environment 
    git checkout Chinonso-Johnson-2021-01-05
    npm install
    npm run dev

### Build App for Production 
    git checkout Chinonso-Johnson-2021-01-05
    npm install
    npm run build
    npm run start

### Test API:
    Run npm test
